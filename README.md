# ATM Banking Management System

This project is a Banking Management System developed using Spring Boot and Java.

## Modules

- auth-service
  - Customer Registration
  - JWT Authentication
  - Deposit
  - Withdraw
  - Fund Transfer
  - Transaction History
  - Search Customer
  - Create Bank Account

- eureka-server
  - Service Discovery

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- MySQL
- Maven
- Eureka Server
